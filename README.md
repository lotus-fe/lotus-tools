## 安装

```
// npm 
npm install --dev @lotus-ui/tools

// yarn 
yarn add --dev @lotus-ui/tools
```

**也可全局安装**

## 使用

* 命令行

```
// 在当前目录下执行

// 创建一个组件
lotus-tools create -n button

// 执行编译 
lotus-tools run build
```
